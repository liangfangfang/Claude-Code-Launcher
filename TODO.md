# 需求实现清单

## 需求列表

1. [x] 支持输入时切换中文输入法，输入中文项目名并成功保存
2. [x] 要能选择中文路径的项目，功能要正常
3. [x] 要能通过交互点击正常新增、编辑、删除模板
4. [x] 要支持项目分组，最多10个，分组直接在主屏新建，用类似页签切换的方式管理，要有全部显示的页签，前部显示时，不同组的项目用分隔线分开上下显示
6. [x] 关于信息要有版本更新内容和版本号变更
7. [x] 要支持连续失效配置，失效时提示将覆盖上一个备份文件，确认后失效并覆盖
8. [x] 关闭按钮要是个X
9. [x] windows要正确显示图标
10. [x] 备份管理和全局配置合并为全局配置管理，应能支持全局配置的读取、修改、保存、应用模板、删除、失效、恢复
11. [x] 启动时检测是否管理员身份启动，如不是应申请管理员身份启动，申请通过则进入软件，申请不通过弹出弹窗提示，确定后应退出
